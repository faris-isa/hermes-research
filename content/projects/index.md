## How to Add a Project

1. Create `content/projects/<project-name>/index.md`
2. Add subfolders: `architecture/`, `modules/`, `knowledge/`
3. Link notes with `[[wikilinks]]`
4. Push to GitHub — Quartz rebuilds automatically
