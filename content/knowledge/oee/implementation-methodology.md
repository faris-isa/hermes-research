# OEE Implementation Methodology

## Phase 1: Foundation (Weeks 1-4)

1. **Define scope:** Which lines/machines first? Start with bottleneck
2. **Select data collection method:** PLC, CV, or hybrid
3. **Set baseline:** 4+ weeks of automatic measurement
4. **Train operators:** Importance, how to log downtime reasons

## Phase 2: Baseline and Analysis (Weeks 5-8)

1. **Establish baseline OEE, A, P, Q** — this WILL be lower than expected
2. **Identify top 3 losses** by category (availability, performance, quality)
3. **Root cause analysis** on each major loss
4. **Set improvement targets** (TCB or AFSM method)

## Phase 3: Improvement (Weeks 9-16)

1. **Implement targeted countermeasures** (SMED, TPM, Poka-Yoke)
2. **Track results** — expect gradual improvement
3. **Expand to additional lines** once proven on bottleneck
4. **Integrate with CMMS** for automated maintenance triggers

## Phase 4: Sustain and Expand (Ongoing)

1. **Daily management reviews** of OEE dashboards
2. **Weekly improvement meetings** analyzing losses
3. **Monthly target adjustments** based on achievement
4. **Annual benchmarking** against industry standards

## Critical Success Factors

| Factor | Why It Matters |
|--------|----------------|
| **Executive Sponsorship** | Resources and organizational commitment |
| **Operator Engagement** | Frontline knowledge drives accurate data + improvements |
| **Automated Data Collection** | Eliminates manual entry errors, provides real-time visibility |
| **Focus on Bottleneck** | Highest ROI on constraint machine |
| **Start Small, Prove Value** | One line first, then expand |
| **Do Not Compare to 85%** | Focus on continuous improvement, not absolute targets |

## Common Implementation Mistakes

1. **Trying to measure everything at once** — start with one line
2. **Skipping operator training** — they have deepest machine knowledge
3. **Using manual data collection** — error-prone and outdated
4. **Setting one target for all lines** — different products need different targets
5. **Ignoring the OEE Drop** — expect 15-20 point drop when switching to automatic measurement
6. **Focusing only on OEE score** — analyze A, P, Q individually to find root causes
